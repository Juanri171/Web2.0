Preguntas y respuestas

AP: D-Link DAP-2360

-- Versió de Wifi: IEEE 802.11n, IEEE 802.11g, IEEE 802.3ab,IEEE 802.3af, IEEE 802.3u, IEEE 802.3.
-- Nombre màxim i recomanat de dispositius connectats alhora: Entre 8 y 15 dispositivos.
-- Preu: 200 €.
-- Velocitat: 300 Mbps.
-- IP de fàbrica: 192.168.0.50
-- Com podem accedir a l’AP des d’un ordinador connectat físicament o per Wifi?
Para acceder al AP las Ip's del mismo dispositivo y la de mi ordenador tienen que estar en la misma subred.

-- Com podem canviar la IP per defecte de l’AP per una que es trobi dins la xarxa:
Accediendo al navegador web y poner la IP 192.168.0.50 ahí accedearán al AP y podran cambiar la IP por defecto.

-- L’AP pot tenir 4 modes de funcionament, quins són? com actuen cada un d’ells?
Modo cliente: Se utiliza como receptor y actúa como un cable de red uniéndose a una red.
AP (punto de Acceso): El Punto de acceso sirve de núcleo para la instalación del cableado, de forma que los múltiples usuarios acceden a la red por medio del punto de acceso.
Modo Repetidor: Este modo se puede usar para extender la señal de forma que el punto de acceso amplifica la señal que recibe para optimizar el rango de acción.
Modo Bridge: Este modo se hace para cubrir grandes distancias, como dos edificios separados. Con dos puntos de acceso conectados entre sí podemos conseguir una red WLAN a distancias considerables.

-- Com podem crear SSID nous? Què son els SSID?
Selecciona el ícono de Wireless Setup en el menú principal:
Si tienes la opción, selecciona "2.4 GHz" o "5 GHz" en la barra lateral izquierda.
Selecciona "SSID Setup" desde el menú en la barra lateral izquierda.
Tu red principal debería aparecer primero en la lista desplegable de SSID.
SSID se trata del nombre que tiene cada red Wi-Fi y sirve para identificarla frente a otras que haya dentro del mismo rango. Por ejemplo si hacemos una búsqueda en nuestra vivienda, nos aparecerá el SSID de nuestra red, así como otras de otros vecinos.

-- Com podem establir un sistema de seguretat per credencials per accedir a l’SSID?
Deberemos configurar un SSID diferente con una contraseña alternativa a las que tenemos, tanto para el acceso al router como para el acceso a la red wifi local.

-- Quin mode de seguretat és el més recomanable, WEP, WPA o WPA2? Per què?
WPA2 porque puedes agregar, editar de una forma más segura y a tu propio estilo de personalización la SSID.

-- Es poden crear diferents VLAN? Com?
Si, con las SSID y se pueden crear hasta 16 VLAN.

-- Com podem accedir al switch des d’un ordinador connectat físicament o per Wifi?

Para acceder al switch las Ip's del mismo dispositivo y la de mi ordenador tienen que estar en la misma subred.

-- Com podem canviar la IP per defecte de l’AP per una que es trobi dins la xarxa?
Accediendo al navegador web y poner la IP 192.168.0.239 ahí accederán al switch y podran cambiar la IP por defecto.

-- Què és el mirroring? Es pot fer en el switch? Com?
El port mirall o port mirroring és utilitzat amb un switch de xarxa per enviar còpies de paquets de xarxa vistos en un port del switch (o una VLAN sencera) a una connexió de xarxa monitoritzada en un altre port del switch.

-- VLAN: s’han de crear 2 subxarxes virtuals (VLAN) tot seleccionant les boques oportunes i després provar-ho:
Para crear 2 subredes virtuales en una VLAN tienes que hacerlo de forma manual desde el switch.

Router:Router TL-ER604W

-- Al Router hi ha dos ports WLAN. Per a que serveixen? Com se configuren? Sirven para crear un red de área local inalámbrica y se configuran manualmente desde el router. A la configuració LAN, hi ha l’adreça IP del Router que és de fet la porta d’enllaç, però també hi ha altres pestanyes del servidor intern DHCP.

-- Com podem configurar un rang d’adreces dinàmiques i un altre d’estàtiques?
Con un servidor DHCP.

-- Per què serveix la pestanya DHCP CLient?
Es una función sirve para comprobar los dispositivos que hay conectados en el router.

-- I la de DHCP Reservation?
Es una función que le permite al router reservar el alquiler de una dirección IP para el uso de un dispositivo específico en la red, garantizando de manera efectiva que el router no asigna la dirección IP a otros dispositivos de la red.

-- Què és el mirroring? Es pot fer en el router? Com?
El port mirall o port mirroring és utilitzat amb un switch de xarxa per enviar còpies de paquets de xarxa vistos en un port del switch (o una VLAN sencera) a una connexió de xarxa monitoritzada en un altre port del switch. En el router se puede hacer configurando manualmente un puerto con ajustes específicos de dúplex y velocidad.

-- Com es poden crear VLAN?
Con les SSID.

-- A la configuració Wifi, com es poden configurar les SSID?
Manualment desde el router.

-- Què és WDS?
Es un sistema de distribució inalámbrica que permet la interconexió inalámbrica de punts de acces (AP) a una red.

-- Com funciona el MAC filtering?
El filtratge MAC funciona donant instruccions al router perquè permeti connectar-se als dispositius el MAC dels quals aparegui en un llistat.
